Name


expand - convert tabs to spaces

Synopsis


expand long-option
expand [-t tablist][file ...]

Description


The expand utility prints files or in none are specified the standard input with tab characters replaces with one or more spaces that are needed to pad to the next tab stop. By default tab stops are 8 spaces apart.

Options


-t tablist Specify the tab stops. This can be either a single value or list seperated by commas or blanks. If a single value is given, tabs are set at that distance apart on each line until the newline character is reached. If a list is given, then each value represents the column that the tab stop is to be set to.
--help Print the help message and exit
--version Print program version and exit
