uniq

Name


uniq - Report or filter out repeated lines in a file

Synopsis


uniq long-option
uniq [-c|-d|-u][-f fields][-s char] [infile [outfile]]

Description


The uniq utility removes duplicate lines from files. Files should be sorted before executing uniq

Options


-c Precede each output line with a count of the number of times the line occured in the input
-d Only write duplicate lines
-f fields [[:blank:]]*[^[:blank:]]*
-s char
-u Do not write lines that are repeated in the input
--help Print the help message and exit
--version Print program version and exit
