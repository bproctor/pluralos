cmp

Name


cmp - Compare two files

Synopsis


cmp long-option
cmp [-l | -s][-i n[:m]] [file1] [file2]

Description


The cmp utility compares two files and tells whether or not they differ. If they do differ, then cmp will report on what line and byte they differ.

Options


-i n[:m] This option allows for ranges to be compared. n says which is the first byte to compare, and m tell the last byte to compare. If no m is given, cmp will skip the first n bytes and compare till the end of the file.

The values can have suffixes on them. These suffixes multiply the value by the amount listed below:
k 1,000 kb 1,024
m 1,000,000 mb 1,048,576
g 1,000,000,000 gb 1,073,741,824
The suffixes can also be in capital letters.

-l Print the bytes that differ from each file
-s Suppresses all output and exits with a status code of zero if the files are the same, otherwise it exits with 1
--help Print the help message and exit
--version Print program version and exit
