du

Name


du - Estimate disk usage

Synopsis


du long-option
du [-a|-s][-kx][-H|-L][file ...]

Description


The du utility estimates the amount of disk used to store the specified files.

Options


-a
-H
-k
-L
-s
-x
--help Print the help message and exit
--version Print program version and exit
