join

Name


join - Relational database operator

Synopsis


join long-option
join [-a fnum | -v fnum][-e string][-o list][-t char][-1 field][-2 field] file1 file2

Description


The join utility performs an equality join of files file1 and file2

Options


-a fnum
-e string
-o list
-t char
-v fnum
-1 field
-2 field
--help Print the help message and exit
--version Print program version and exit
