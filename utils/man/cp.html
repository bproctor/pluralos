cp

Name


cp - Copy files

Synopsis


cp long-option
cp [-firv] src ... dest

Description


The cp utility copies files. All files listed on the command line are copied to the last file, dest. If the -r option is specified, then the listed files can be directory and all of there contents will be copied.

Options


-f Force
-i Interactive
-r Recursive
-v Verbose
--help Print the help message and exit
--version Print program version and exit
