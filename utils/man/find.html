find

Name


find - Find files

Synopsis


find long-option
find [-H | -L] path ... expression ...

Description


The find utility recursively descends the directory tree and prints each the path for each file that it finds that meet the expression specified to the standard output.

Options


-H Dereference symbolic links
-L
--help Print the help message and exit
--version Print program version and exit
The following primaries for the expression argument are supported:
-atime n
-ctime n
-depth
-exec name [arg ...] ; -exec name [arg ...] { } +
-group gname
-links n
-mtime n
-name pattern
-newer file
-nouser
-nogroup
-ok name [arg ...] ;
-perm [-]mode
-perm [-]onum
-print
-prune
-size n[c]
-type c
-user uname
-xdev
