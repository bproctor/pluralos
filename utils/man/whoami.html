whoami

Name


whoami - print the current users login name

Synopsis


whoami [long-option]

Description


whoami will print the current user's login name that is associated with the currect effective ID. This is the same as the running `who am i` or `id -un`.

Options


--help Print the help message and exit
--version Print program version and exit
