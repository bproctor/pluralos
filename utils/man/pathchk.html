pathchk

Name


pathchk - Check a path for validity

Synopsis


pathchk long-option
pathchk [-p] pathname ...

Description


The pathchk utility checks one or more pathnames for validity. A valid pathname will be one that could be used to access or create a file. pathchk also checks for portability so that no filename truncation would result in using the pathname.

Options


-p Checks the pathname for invalid characters and length
--help Print the help message and exit
--version Print program version and exit
