pwd

Name


pwd - Print the current working directory

Synopsis


pwd long-option
pwd [-L | -P]

Description


The pwd utility prints the current working directory to the standard output.

Options


-L If the PWD environment variable contains a pathname for the current working directory that does not contain . or .. pwd prints this pathname to the standard output. Otherwise, this option behaves as the -P option
-P Do not write a pathname that contains symbolic links
--help Print the help message and exit
--version Print program version and exit
If both -L and -P are specified, the last one applies. If neither are specified, then pwd behaves as if -L was specified.
