nl

Name


nl - Line numbering filter

Synopsis


nl long-option
nl [-p][-b type][-d delim][-f type][-h type][-i incr][-l num][-n format][-s sep][-v startnum][-w width][file]

Description


The nl utility numbers the lines of the input files or standard input if no input files are specified.

Options


-b type Set the body style for line numbering to type
-d delim Specify the delimiter that character that indicates a new logical page
-f type Set the footer style for line numbering to type
-h type Set the header style for line numbering to type
-i incr Increment the line number by incr for each line
-l num
-n format Set the format to print the line numbers to format
-p Restart line count each time a new page is encountered
-s sep
-v startnum
-w width Set the width of the line numbering column to width
--help Print the help message and exit
--version Print program version and exit
