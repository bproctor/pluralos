chown

Name


chown - change the file ownership

Synopsis


chown long-option
chown -hR owner[:group] file ...
chown -R [-H | -L | -P] owner[:group] file ...

Description


The chown utility

Options


-h Chang IDs of symbolic links, not the file that the symbolic link links to.
-R Recursively change file IDs. With this option chown will traverse direcories that are specified and change all the IDs for files found in those directories.
-H
-L
-P
--help Print the help message and exit
--version Print program version and exit
