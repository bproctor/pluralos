shred

Name


shred - Overwrite files to make them difficult to recover

Synopsis


shred long-option

Description


The shred utility is used to remove sensitive data by overwriting the files specified multiple times before deleting them so that recovering them is very difficult, if not impossible.

Options


--help Print the help message and exit
--version Print program version and exit
