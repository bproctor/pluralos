mkfifo

Name


mkfifo - make FIFO special files

Synopsis


mkfifo long-option
mkfifo [-m mode] file ...

Description


The mkfifo utility is used to create FIFO special files.

Options

-m mode Sets the file permission bits of the FIFO to mode, as in chmod
--help Print the help message and exit
--version Print program version and exit
