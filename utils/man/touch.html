touch

Name


touch - Change a files modification time

Synopsis


touch long-option
touch [-acm][-r ref-file | -t time] file ...

Description


The touch utility changes the modification time of files.

Options


-a
-c
-m
-r ref-file
-t time
--help Print the help message and exit
--version Print program version and exit
