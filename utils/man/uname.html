uname

Name


uname - Print information about the system

Synopsis


uname long-option
uname [-amnrsv]

Description


uname prints information about the system. If no option is specified, then uname behaves as if the -s option was used.

Options


-a Behave as if all options (-mnrsv) had been specified.
-m Print the machine type the system is running on
-n Print the name of this node.
-r Print the release level of the operating system
-s Print the system name of the operating system
-v Print the version level of the operating system
--help Print the help message and exit
--version Print program version and exit
