cut

Name


cut - Cut out parts of a file

Synopsis


cut long-option
cut -b list [-n] [file ...]
cut -c list [file ...]
cut -f list [-d delim][-s][file ...]

Description


The cut utility cuts out parts a file or files.

Options


-b list
-c list
-d delim
-f list
-n
-s
--help Print the help message and exit
--version Print program version and exit
