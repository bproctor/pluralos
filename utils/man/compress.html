compress

Name


compress - Compress a file

Synopsis


compress long-option
compress

Description


The SUSV3 standard states that the compress utility use the Lempel-Ziv compression algorithm. Lempel-Ziv is US Patent 4464650 and therefore is not supported in the Plural Operating System. Use pzip instead.

Options


--help Print the help message and exit
--version Print program version and exit
