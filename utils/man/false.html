Name


false - Do nothing, exit failure

Synopsis


false [long-option]

Description


The false utility does nothing but exit's as failure when called. It can be used in shell scripts where exiting as failure is necessary. Note that if a long option is used false will exit successfully.

Options


--help Print the help message and exit
--version Print program version and exit
