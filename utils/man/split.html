split

Name


split - Split a file into smaller pieces

Synopsis


split long-option split [-l line-count] [-a suffix-length] [file[name]]
split -b n[klm] [-a suffix-length] [file[name]]

Description


The split utility splits the specified file into smaller pieces.

Options


-a suffix-length
-b nk
-b nl
-b nm
-l line-count
--help Print the help message and exit
--version Print program version and exit
