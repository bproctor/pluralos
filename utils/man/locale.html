locale

Name


locale - Get locale specific information

Synopsis


locale long-option
locale [-a | -m]
locale [-ck] name ...

Description


The locale utility prints information about the current locale environment.

Options


-a
-c
-k
-m
--help Print the help message and exit
--version Print program version and exit
