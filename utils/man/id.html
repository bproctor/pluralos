id

Name


id - Print information about the specified user

Synopsis


id long-option
id [-agGnru] [user ...]

Description


The id utility prints information about the specified user to the standard output. If no user is specified the current user is used.

Options


-a
-g Print the effective group ID
-G Print all group IDs
-n Print a name instead of a number
-r Print the real ID instead of the effective ID
-u Print the effective user ID
--help Print the help message and exit
--version Print program version and exit
