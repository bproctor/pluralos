cat

Name


cat - concatenate files.

Synopsis


cat long-option
cat [-u][file ...]

Description


The cat utility reads files in sequence and writes the contents of the files to the standard output in the same sequence. If '-' is given in the file list, cat will read from the standard input.

Options


-u Unbuffered, writes bytes without delay as each byte is read
--help Print the help message and exit
--version Print program version and exit
