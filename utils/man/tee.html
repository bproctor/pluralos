tee

Name


tee - Read from standard input and write to standard output

Synopsis


tee long-option
tee [-ai][file...]

Description


The tee utility reads from the standard input and writes to the standard output or files.

Options


-a Append to the given file, do not overwrite
-i Ignore signals
--help Print the help message and exit
--version Print program version and exit
