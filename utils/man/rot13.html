Name


rot13 - Encrypt or decrypt a file with a simple rotate-13 algorithm

Synopsis


rot13 long-option
rot13

Description


The rot13 utility encrypts a text file with a rotate-13 algorithm. Please note that this algorithm is not secure and easily breakable. Do not use this utility with sensitive data.

Options


--help Print the help message and exit
--version Print program version and exit
