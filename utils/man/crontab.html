crontab

Name


crontab - Schedule jobs for later execution

Synopsis


crontab long-option
crontab [-u user] [-e | -l | -r]

Description


The crontab utility schedules jobs for later execution.

Options


-e Edit the crontab entry. If no entry exists, create a new one to edit.
-l List the user's crontab entries
-r Remove the user's crontab entry
-u user Specify which user's crontab entry to use
--help Print the help message and exit
--version Print program version and exit
