tsort

Name


tsort - Topological sort

Synopsis


tsort long-option
tsort [file]

Description


The tsort utility prints to the standard output an ordered list of items consistent with a partial ordering of items contained in the input. If no input file is specified, the standard input is used.

Options


--help Print the help message and exit
--version Print program version and exit
