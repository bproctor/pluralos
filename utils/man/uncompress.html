uncompress

Name


uncompress - Uncompress a file

Synopsis


uncompress long-option
uncompress

Description


The SUSV3 standard states that the uncompress utility use the Lempel-Ziv compression algorithm. Lempel-Ziv is US Patent 4464650 and therefore is not supported in the Plural Operating System. Use punzip instead.

Options


--help Print the help message and exit
--version Print program version and exit
