seq

Name


seq - Print numbers in sequence

Synopsis


seq long-option
seq [-f fmt] [-s sep] last
seq [-f fmt] [-s sep] first last
seq [-f fmt] [-s sep] first inc last

Description


The seq utility prints to the standard output a sequence of numbers.

Options


-f fmt Use the printf style format specifier to output numbers
-s sep Use sep as a seperator between numbers. The default is the newline character
--help Print the help message and exit
--version Print program version and exit
