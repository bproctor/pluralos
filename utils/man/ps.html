ps

Name


ps - Report process status

Synopsis


ps long-option
ps [-aAdefl][-G grouplist][-o format]...[-p proclist][-t termlist][-U userlist] [-g grouplist][-n namelist][-u userlist]

Description


The ps utility prints a report of the status of processes.

Options


-a Print information about all processes that are associated with terminals
-A Print information about all processes
-d Print information about all processes except session leaders
-e Print information about all processes, this option is equivalent to -A
-f Print full-format output
-g grouplist
-G grouplist
-l Print long-format output
-n namelist
-o format Specifies the format for output
-p proclist Print information about all processes whose PID is given in proclist
-t termlist
-u userlist Print information about all processes whose UID is given in userlist
-U userlist Print information about all processes whose UID is given in userlist. This is the same as the -u option except that it does not output the UID in the listing
--help Print the help message and exit
--version Print program version and exit
