iconv

Name


iconv - Codeset conversion

Synopsis


iconv long-option
iconv [-cs] -f fromcode -t tocode [file ...]
iconv -l

Description


The iconv utility converts the characters from file from one codeset to another and writes the results to that standard output.

Options


--help Print the help message and exit
--version Print program version and exit
