expr

Name


expr - Calculate an expression

Synopsis


expr long-option
expr [-f] format

Description


The expr utility calculates an expression given on the command line. Expressions can be built using the following operators. If the -f option is used, some of these operands are not available.
expr1 + expr2 Add expr1 and expr2
expr1 - expr2 Subtract expr2 from expr1
expr1 * expr2 Multiply expr1 and expr2
expr1 / expr2 Divide expr1 by expr2
expr1 % expr2 Find remainder of expr1 divided by expr2
expr1 << expr2 Shift expr1 left by expr2 bits
expr1 >> expr2 Shift expr1 right by expr2 bits
expr1 ** expr2 Raise expr1 to the power of expr2
expr1 == expr2 If expr1 is equal to expr2, 1, else 0
expr1 ^ expr2 Bitwise Exclusive OR
expr1 | expr2 Bitwise OR
expr1 & expr2 Bitwise AND
expr1 && expr2 Logical AND
expr1 || expr2 Logical OR
expr1 != expr2 If expr1 is not equal to expr2, 1, else 0
expr1 < expr2 If expr1 is less than expr2, 1, else 0
expr1 > expr2 If expr1 is greater than expr2, 1, else 0
expr1 <= expr2 If expr1 is less than or equal to expr2, 1, else 0
expr1 >= expr2 If expr1 is greater than or equal to expr2, 1, else 0
(expr) Group an expression for precedence
-expr Negate expr
!expr Logical not of expr
~expr Compliment of expr

Options


-f Use floating point numbers
--help Print the help message and exit
--version Print program version and exit
