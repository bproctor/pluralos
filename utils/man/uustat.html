uustat

Name


uustat - uucp status inquery and job control

Synopsis


uustat long-option
uustat [-q | -k jobid | -r jobid]
uustat [-s system][-u user]

Description


The uustat utility displays the status, or cancels a previous uucp request. It can also provide general status on uucp connections to other systems.

If no option is given, uustat will write the status of all uucp requests issued by the current user to the standard output.

Options


-k jobid Kill the uucp request with job identifier jobid.
-q Print the jobs queued for each machine
-r jobid Rejuvenate jobid. This changes the modification time of files associated with jobid to the current time so that the cleanup program does not delete the job until the jobs identification time reaches the limit imposed by the program.
-s system Print the status of all uucp requests for remote system system
-u user Print the status of all uucp requests issued by user
--help Print the help message and exit
--version Print program version and exit
