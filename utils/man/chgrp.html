chgrp

Name


chgrp - Change the file group ownership

Synopsis


chgrp long-option
chgrp -hR group file ...
chgrp -R [-H | -L | -P] group file ...

Description


chgrp changes the group ID of the files specified. If more than one of -H, -L, or -P is specified, the last one is choosen by chgrp.

group This can be either a group name or a group ID number.
file one or more files can be entered on the command line. If this argument is a directory, you can use the -R option to traverse the directory and change the group ownership of each file within the directory.

Options


-h Chang group IDs of symbolic links, not the file that the symbolic link links to.
-R Recursively change file group IDs. With this option chgrp will traverse direcories that are specified and change all the group IDs for files found in those directories.
-H
-L
-P
--help Print the help message and exit
--version Print program version and exit
