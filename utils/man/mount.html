mount

Name


mount - Mount a filesystem

Synopsis


mount long-option
mount [-l]
mount -a [-t | -O] ...
mount [-t type] [device] [directory]

Description


The mount utility mounts a filesystem. If no arguments are given mount will print a list of the currently mounted filesystems.

Options


-a Mount all filesystems specified in the /etc/fstab file
-O
-t type Specify the type of filesystem to mount
--help Print the help message and exit
--version Print program version and exit
