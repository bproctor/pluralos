tabs

Name


tabs - Change tabs on terminal

Synopsis


tabs long-option
tabs [-a|-a1|-a2|-a3|-c|-c2|-c3|-C|-C2|-C3|-C4|-f|-n|-p|-s|-u][+m[n]][-T type]
tabs [-T type][+[n]] n1[,n2,...]

Description


The tabs utility sets the tabs on the terminal.

Options

-a Assembler, (1, 10, 16, 36, 72)
-a1 Assembler, x86, (1, 9, 17, 25, 33, 41, 49, 57, 65, 73)
-a2 Assembler, (1, 10, 16, 40, 72)
-a3 Assembler, (...)
-c COBOL, (1, 8, 12, 16, 20, 55)
-c2 COBOL, (1, 6, 10, 14, 49)
-c3 COBOL, (1, 6, 10, 14, 18, 22, 26, 30, 34, 38, 42, 46, 50, 54, 58, 62, 67)
-C C, C++ (Incomplete)
-C2 C, C++, ()
-C3 C, C++, ()
-C4 C, C++, ()
-f Fortran, (1, 7, 11, 15, 19, 23)
-n
-p PL/1, (1, 5, 9, 13, 17, 21, 25, 29, 33, 37, 41, 45, 49, 53, 57, 61)
-s SNOBOL, (1, 10, 55)
-T type Specify the terminal type
-u Assembler, (1, 12, 20, 44)
--help Print the help message and exit
--version Print program version and exit
