md5sum

Name


md5sum - Print or check a MD5 (128-bit) checksum

Synopsis


md5sum long-option
md5sum [-bcstw] file ...

Description


The md5sum utility prints a checksum.

Options


-b
-c
-s
-t
-w
--help Print the help message and exit
--version Print program version and exit
