tac

Name


tac - Print files in reverse

Synopsis


tac long-option
tac [-s string] file ...

Description


The tac utility prints files in reverse line order.

Options


-s string Use string as a seperator instead of the newline character
--help Print the help message and exit
--version Print program version and exit
