mknod

Name


mknod - Make /dev entry special files

Synopsis


mknod long-option
mknod [-m mode] name type [major] [minor]

Description


The mknod utility make /dev entry special files

name The name of the special file to create
type The type of special file to create
major The major number for the special file
minor The minor number for the special file

Options


-m mode Sets the permissions bits of the special file to mode, as in chmod
--help Print the help message and exit
--version Print program version and exit
