Name


asa - Interpret carriage-control characters

Synopsis


asa long-option
asa [file ...]

Description


The asa utility writes it's input files to the standard output while converting carriage-control characters into line-printer control sequences. If no file is specified on the command line, the standard input will be used.

asa uses the first character of each line to determine the output. The first character can be one of the following:

space The rest of the line is output without change.
0 A newline character is output, and then the rest of the line.
1 Advances to the next page, then the rest of the line.
+ Go to the beginning of the previous line, then the rest of the line. If + is the first character from the input, it is equivalent to .

Options


--help Print the help message and exit
--version Print program version and exit
