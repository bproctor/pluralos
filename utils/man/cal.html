cal

Name


cal - print a calender

Synopsis


cal long-option
cal [[month] year]

Description


The cal program prints a month calender for the given month and year. If only a year is specified, then cal will print a calender for the month if January of the given year. If neither month nor year is given, cal will print a calender of the current month and year.

month can be written as a number between 1 and 12, or as the word (January, February, March, April, May, June, July, August, September, October, November, Decemeber) or as an abbreviated word (Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec).

year must be the complete year. Two digit year abreviations are not supported. i.e 85 is not 1985, but instead the year 85. year can be any number between 1 and 9999.

Options


--help Print the help message and exit
--version Print program version and exit
