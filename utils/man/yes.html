yes

Name


yes - print `y' or a message forever

Synopsis


yes long-option
yes [msg]

Description


yes prints the message 'msg', or if no message is given prints 'y' forever until killed. yes can be killed from the terminal by using the Ctrl-C key combination.

Options


--help Print the help message and exit
--version Print program version and exit
