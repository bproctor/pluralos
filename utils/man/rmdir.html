rmdir

Name


rmdir - Remove directories

Synopsis


rmdir long-option
rmdir [-fipv] dir ...

Description


The rmdir utility removes the each directory that is specified on the command line. The directories must be empty to remove them.

Options


-f Force
-i Interactive
-p
-v Verbose
--help Print the help message and exit
--version Print program version and exit
