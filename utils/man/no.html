Name


no - print `n' or a message forever

Synopsis


no long-option
no [msg]

Description


no prints the message 'msg', or if no message is given prints 'y' forever until killed. no can be killed from the terminal by using the Ctrl-C key combination.

Options


--help Print the help message and exit
--version Print program version and exit
