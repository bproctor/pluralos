link

Name


link - Link files using the link function

Synopsis


link long-option
link file1 file2

Description


The link utility uses the link() function to link files together. file1 should be the existing file, while file2 is the file that is to be created as a link.

Options


--help Print the help message and exit
--version Print program version and exit
