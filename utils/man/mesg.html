mesg

Name


mesg - Allow or deny message

Synopsis


mesg long-option
mesg [y|n]

Description


The mesg utility specifies whether to allow or deny messages that are sent to the terminal by other users. If no argument is given, mesg will report the current state. Note that processes with appropriate privileges (i.e. root) will still be able to send message to the terminal regardless of the state.

Options


y Allow other users to send messages to the terminal
n Deny permission for other users to send messages to the terminal
--help Print the help message and exit
--version Print program version and exit
