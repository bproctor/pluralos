fold

Name


fold - Fold long lines to the specified length

Synopsis


fold long-option
fold [-bs][-w width][file ...]

Description


The fold utility breaks long lines from file or the standard input to the specified length and writes the results to the standard output. The default for the width of a line is 80.

Options


-b Count width in bytes rather than column positions
-s Only break lines on blanks
-w width Specify the maximum line length
--help Print the help message and exit
--version Print program version and exit
