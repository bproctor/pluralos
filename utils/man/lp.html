lp

Name


lp - Send finds to the printer

Synopsis


lp long-option
lp [-c][-d dest][-n copies][-msw][-o option]...[-t title][file ...]

Description


The lp utility sends files to the printer.

Options


-c
-d dest
-m
-n copies
-o option
-s
-t title
-w
--help Print the help message and exit
--version Print program version and exit
