Name


wc - count words, lines, and characters/bytes

Synopsis


wc long-option
wc [-c | -m] [-lw] file ...

Description


The wc utility

Options


-c Count characters
-m Count bytes
-l Count lines
-w Count words
--help Print the help message and exit
--version Print program version and exit
