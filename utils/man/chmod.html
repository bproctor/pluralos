chmod

Name


chmod - Change file permisions

Synopsis


chmod long-option
chmod mod files ...

Description


The chmod utility changes the permission bits described in mod of the file file.

mod can be described in two ways, the first is a set of four octal digits. If the first of the octal digits is not specified, it is assumed to be 0. The following three are required. They are created by adding up the desired permissions as descibed below. The synopsis for this method can be described as [O]OOO where O is an octal digit
1 Read permission
2 Write permission
4 Execute permission

The set of octal digits is as follows:
First digit
Second digit Describes permissions given to the owner of the file
Third digit Describes permissions given to the group the owner belongs to
Fourth digit Describes permissions given to everyone else

The second way to descibe the permission bits is using a letter to descibe a set of users, followed by +, - or = to add, remove, or set permission, followed by a combination of letters descibing what permission to change. For example, `g=rw' sets permission so the group has read and write permissions. `o-x' removes execute permission from others but any other permissions that may exists are not affected. The first letter can be one of the following:
u The user that owns the file
g The group the owner belongs to
o Everyone else who is not the owner and not a member of the group the owner belongs to

The last part can be made up of one or more of the following letter:
r Read permission
w Write permission
x Execute permission

Options


--help Print the help message and exit
--version Print program version and exit
