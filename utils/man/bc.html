bc

Name


bc - Arbitrary precision arithmetic language

Synopsis


bc long-option
bc [-l] [file ...]

Description


The bc utility reads from the files specified, and implements an arbitrary calculator. If no files are specified, the standard input is used and bc becomes interactive.

bc supports the following keywords:
auto
break
ibase
if
for
length
obase
quit
return
scale
sqrt
while

Options


-l Define the math functions and initialize scale to 20, instead of the default of 0
--help Print the help message and exit
--version Print program version and exit
