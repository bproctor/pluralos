basename

Name


basename - strips directory and suffix from filenames.

Synopsis


basename long-option
basename name [suffix]

Description


The basename utility prints name with any leading directory components removed to the standard output. If specified, basename will also remove a trailing suffix.

Options


--help Print the help message and exit
--version Print program version and exit
