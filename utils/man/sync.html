sync

Name


sync - Sync the cached memory with the hard disk

Synopsis


sync [long-option]

Description


The sync utility syncs the cached memory with the hard disk. This utility is useful for make sure that data still in memory is written to that hard disk before shutting down the system.

Options


--help Print the help message and exit
--version Print program version and exit
