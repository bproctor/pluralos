domainname

Name


domainname - Print or change the system domainname

Synopsis


domainname long-option
domainname [name]

Description


The domainname utility prints or changes the system domainname. If no name is given on the command line, the system domainname is printed. Otherwise, the domainname is changed to the name given on the command line, name.

Options


--help Print the help message and exit
--version Print program version and exit
