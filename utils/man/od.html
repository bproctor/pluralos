od

Name


od - Print files in different formats

Synopsis


od long-option
od [-v][-A address-base][-j skip][-N count][-t type-string]...[file ...]

Description


The od utility prints files in different formats.

Options


-A address-base
-j skip
-N count
-t type-string
-v
--help Print the help message and exit
--version Print program version and exit
