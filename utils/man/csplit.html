csplit

Name


csplit - split files base on context

Synopsis


csplit long-option
csplit [-ks][-f prefix][-n num] file arg1 ... argn

Description


The csplit utility read from file and writes all or part of the file into the other files specified by the arg operands, the writes the sizes of the files. If an error occurs, the created files are deleted.

Options


-f prefix Set the prefix for the file names to be created to prefix. The default prefix is xx, which would created the files xx00, xx01, ... xxnn
-k Leave created files intact if an error occurs
-n num Use num digits to create file names
-s Suppress output of file sizes
--help Print the help message and exit
--version Print program version and exit
