mkdir

Name


mkdir - make directories

Synopsis


mkdir long-option
mkdir [-p][-m mode] dir ...

Description


The mkdir utility makes directories.

Options


-m mode Set the file permissions of the directory to mode
-p Create any missing intermediate pathname components
--help Print the help message and exit
--version Print program version and exit
