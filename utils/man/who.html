who

Name


who - Print who is on the system

Synopsis


who [long-option]
who [-abdilmpqrstuwHLT] [file | arg1 arg2]

Description


who prints who is logged onto the system.

Options


-a Same as -bdLprtTu
-b Time of last system boot
-d Print dead processes
-i Add idle time as 'hours:minutes'
-l Attempt to canonicalize hostnames via DNS
-m Only hostname and user associated with standard input
-p Print active processes spawned by init
-q All login names and number of users logged in
-r Print current runlevel
-s Print only name, line and time
-t Print last system clock change
-u List users logged in
-w Add users message status as +, -, or ?
-H Print line of column headings
-L Print system login processes
-T Same as -w
--help Print the help message and exit
--version Print program version and exit
