mv

Name


mv - Move files

Synopsis


mv long-option
mv [-firv] src ... dest

Description


The mv utility moves files. The last file listed on the command line is used for the destination.

Options


-f
-i
-r
-v
--help Print the help message and exit
--version Print program version and exit
