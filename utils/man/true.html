Name


true - Do nothing, exit successfully

Synopsis


true [long-option]

Description


The true utility does nothing but exit's successfully when called. It can be used in shell scripts where a successful exit is needed in all cases.

Options


--help Print the help message and exit
--version Print program version and exit
