echo

Name


echo - Print arguments

Synopsis


echo long-option echo [-en] args...

Description


The echo utility prints each of it's arguments to the standard output with a space between each and a newline character at the end.

Options


-e Interpret control characters. The control characters allowed are:
'\a'??? character
'\b'??? character
'\e'Escape character
'\f'Form-feed character
'\n'Newline character
'\r'Carriage return character
'\t'Tab character
'\v'Vertical tab character
'\xDD'Output the character for the hexidecimal value of DD. Hexidecimal value must be two digits, leading 0's are used for smaller values.
'\0OOO'Output the character for the octal value of OOO. Octal value must be three digits, leading 0's are used for smaller values
Any other character following the '\' will be output as it would be if no '\' character were there.
-n Do not output a final newline
--help Print the help message and exit
--version Print program version and exit
