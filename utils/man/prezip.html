prezip

Name


prezip - Encodes certain files for better compression.

Synopsis


prezip long-option
prezip [-cd] infile ... [outfile]

Description


The prezip utility prepares certain files for better compression by writing the difference between bytes. This is useful for getting better compression from multimedia file.

If no input file (infile) is specified the standard input is used. If the -c option is specified, then all files specified on the command line will be assumed to be input files. Otherwise, the last file on the command line will be used as the output file (outfile). If only one file is specified on the command line, it is used as an input file and the standard output is used. You can also use - in the file list to read from the standard input and write to a file.

Options


-c Write to standard output
-d Decode
--help Print the help message and exit
--version Print program version and exit
