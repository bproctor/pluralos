Name


df - Report disk usage of mounted partitions

Synopsis


df long-option
df [-ahHiklpTx][-b size][-t type][-x type]

Description


The df utility prints a report of the amount of disk space used by each partition that is mounted on the system.

Options


-a Include filesystems having 0 blocks
-b size use size size blocks
-h Print sizes in human readable formats, with multiples of 1,024
-H Print sizes in human readable formats, with multiples of 1,000
-i Print inode information instead of block information
-k Use 1kb (1,024 byte) block size
-l Limit listings to local filesystems
-t type Limit listings to filesystems of type type
-T Include filesystem type in listings
-x type Do not list filesystems of type type
--help Print the help message and exit
--version Print program version and exit
